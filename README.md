# NG2 Slider

An Simple Angular2 Slider Component

## Install

```
npm install nextzy/ng2-slider
```

## Usage

Import slider module into you application

```
import { Ng2SliderModule } from 'nextzy/ng2-slider';
```

Zero Configuration

```
<ng2-slider
	[max]="0"
	[min]="100"
	[model]=""
></ng2-slider>
```
